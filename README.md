2. Go to [netlify.com](https://netlify.com)
3. Import your repository
4. Build command: `npm run build`
5. Publish directory: `dist`
6. Click "Deploy"

### Deploy to GitHub Pages

1. Update `vite.config.js` with your repo name:
```javascript
base: '/your-repo-name/',
```

2. Run:
```bash
npm run deploy
```
